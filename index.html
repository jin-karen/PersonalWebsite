About Me

Hello! My name is Karen Jin and I'm a first-generation college graduate from Boston University, with a Bachelors of Science in Hospitality Administration. I delight in the ability to create personalized and memorable experiences for myself and others. Through practical work and classroom learning, I was able to develop my communication skills, global perspective, problem solving capacities, and knowledge in a dynamic industry.

I discovered my love for coding when taking a Python course after graduating. My passion and dedication has since flourished while applying my knowledge to projects as I learned HTML, CSS, and currently Javascript. I aspire to integrate my hospitality background with my continuing programming journey to exceed expectations for both internal and external stakeholders as a software engineer.

In my free time, I also enjoy hobbies such as baking, drawing, playing video games, exploring new places, and trying new foods.

A classic children’s game, to be played by one or two players, with the goal of uncovering a word by guessing letters within a certain number of tries. This game is recreated with Python, to be run in a shell, and it utilizes important concepts such as Object-Oriented Programming.



Shoop Article

A personal, satirical opinion piece that reflects why the singular tense of sheep should be shoop, complete with enticing visuals and a supporting table. This project was created to practice and illustrate knowledge of HTML and CSS elements in a lighthearted and entertaining way.



Weather Report

An informational reporting script that interacts with the Open Weather App API to provide current weather data and daily forecasts based on user-inputted locations (using the API’s geocoding). This Python Script can be handy for everyday usage or to aid with planning for upcoming trips.
